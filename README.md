# Evolve-estadistica-ricardo-guerrero

Repositorio de la práctica final del modulo de Estadística.

## Carpeta principal de la entrega

La entrega evaluable está en:

- practica_final_guerrero_cabrera_ricardo

Dentro de esa carpeta se encuentran:

- Scripts: ejercicio1_descriptivo.py, ejercicio2_inferencia.py, ejercicio3_regresion_multiple.py, ejercicio4_series_temporales.py
- Respuestas: Respuestas.md
- Datos: data/
- Resultados: output/

## Requisitos

- Python 3.12+
- Dependencias en requirements.txt

## Ejecución rápida

Desde la raíz del repositorio:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

cd practica_final_guerrero_cabrera_ricardo
python3 ejercicio1_descriptivo.py
python3 ejercicio2_inferencia.py
python3 ejercicio3_regresion_multiple.py
python3 ejercicio4_series_temporales.py
```
